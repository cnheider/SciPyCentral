About

SciPy Central is a collection of code snippets, modules, links related to scipy stack. It aims to be a central place for SciPy stack users for everything to do with it.

Can I use your content?

The content in this website & including the application is open source licensed. Please refer to our license for more details

How can I contribute to the project?

We're glad that you asked. You can contribute to SciPy Central in several ways.

  • You can share your amazing SciPy/NumPy related content on our application
  • Report application bugs in issue tracker
  • Submit code enhancements, documentation to our application's software
  • If you are not sure where to start, email to our mailing list scipy-dev@scipy.org

Why we made this project?

Back in October 2010 and again in April 2011 there were discussions on the SciPy mailing list about creating a File Central website. This site came about from all the interesting ideas discussed in those threads.
